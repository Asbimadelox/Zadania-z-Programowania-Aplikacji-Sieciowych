Wprowadzenie

Weryfikacja adresu IP
	Napisz program, który sprawdza czy dany adres IP jest poprawnym adresem	

	
Znajdź adres IP znając hostname
	Napisz program, który dla danego hostname wyświetla odpowiadający mu adres IP


Znajdź hostname znając adres IP
	Napisz program, który dla danego adresu IP wyświetli odpowiadający mu hostname.


Połącz się z serwerem przy użyciu TCP
	Napisz program, w którym połączysz się z danym serwerem na określonym porcie przy użyciu protokołu TCP,
	a następnie wyświetlisz informację czy udało Ci się nawiązać połączenie. Program powinien akceptować 
	adres w postaci adresu IP jak i hostnam	


Skaner portów
	Napisz program - skaner portów, który dla danego serwera będzie sprawdzał jakie porty są otwarte przy
	użyciu protokołu TCP. Program powinien akceptować adres w postaci adresu IP oraz hostname
